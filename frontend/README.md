# Insightor Frontend

React + Vite frontend for the Insightor AI Research Assistant.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The frontend will run on `http://localhost:3000` and automatically proxy API requests to the backend on `http://localhost:8000`.

## Build

```bash
npm run build
```

## Features

- Beautiful, responsive UI with Framer Motion animations
- Real-time search input with suggestions
- Animated loading states
- Graceful error handling
- Displays research results with source citations
- Top insights and recommendations from AI analysis
